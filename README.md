# أذكار — Athkar PWA 📿

تطبيق ويب تقدّمي (PWA) للأذكار **الصحيحة الموثّقة بمصادرها** من القرآن والسنة.
مجاني، مفتوح المصدر، يعمل **بدون إنترنت**، ولا يحتاج تثبيتًا من متجر.

> نية المشروع: **صدقة جارية**. مَن نشره أو طوّره أو علّمه، فله أجره بإذن الله.

---

## ✨ المزايا

- **66 ذكرًا صحيحًا** في 6 فئات: الصباح، المساء، بعد الصلاة، النوم، الاستيقاظ، الباقيات الصالحات.
- كل ذكر موثّق بـ **المصدر والفضل وعدد التكرار**.
- **عدّاد باللمس** ينقص مع كل ضغطة + اهتزاز خفيف، ويُحفظ تقدّمك اليومي تلقائيًا.
- **يعمل بدون إنترنت** بالكامل (Service Worker).
- **قابل للتثبيت** على الجوال والحاسوب كأيقونة مستقلة.
- **وضع ليلي** ودعم كامل للعربية (RTL) وخط قرآني جميل.
- **زر مشاركة** لنشر التطبيق بضغطة واحدة.
- خفيف جدًا، بلا حسابات، بلا إعلانات، بلا تتبّع.

## 📁 البنية

```
ATHKAR/
├── index.html              # الواجهة
├── styles.css              # التصميم (فاتح/داكن)
├── app.js                  # المنطق (العدّاد، الحفظ، المشاركة)
├── data/adhkar.js          # قاعدة بيانات الأذكار + المصادر
├── manifest.webmanifest    # تعريف التطبيق (PWA)
├── sw.js                   # العمل بدون إنترنت
└── icons/                  # الأيقونات (SVG + PNG) + مولّدها
```

## 🚀 النشر (مجاني — لأوسع انتشار)

التطبيق ملفات ثابتة فقط، يُرفع على أي استضافة مجانية:

**الأسهل — Netlify Drop:** اذهب إلى <https://app.netlify.com/drop> واسحب مجلد `ATHKAR` كله. يصبح لديك رابط فورًا.

**GitHub Pages:**
```bash
git init && git add . && git commit -m "أذكار المسلم"
# أنشئ مستودعًا على GitHub ثم:
git remote add origin <repo-url> && git push -u origin main
# فعّل Pages من Settings → Pages → Branch: main
```

> ملاحظة: لتثبيت التطبيق والعمل دون إنترنت، يجب أن يُفتح عبر **HTTPS** (وكل المنصّات أعلاه توفّره مجانًا). فتح `index.html` مباشرة من القرص يعرض المحتوى لكن دون ميزة الـ PWA.

## 🔧 التشغيل محليًا

```bash
# أي خادم ثابت، مثال بايثون:
python -m http.server 8000
# ثم افتح http://localhost:8000
```

## ➕ إضافة أذكار

افتح [data/adhkar.js](data/adhkar.js) وأضف عنصرًا للفئة المناسبة بالصيغة:
```js
{ text: "نص الذكر", count: 3, virtue: "فضله", source: "المصدر" }
```
**شرط القبول:** صحة الذكر وثبوت مصدره. لا تُضف إلا ما صحّ.

## 📜 المصادر

القرآن الكريم، صحيح البخاري ومسلم، السنن الأربع، وكتاب **حصن المسلم** للشيخ سعيد بن علي القحطاني، مع الإشارة لتصحيح الألباني حيث ورد.
